## Game and generated data

The displayed game data and game-language translations are generated from
[CrimsonCrossBunker/Cataclysm-Cleanwater-Bomb](https://github.com/CrimsonCrossBunker/Cataclysm-Cleanwater-Bomb),
which is derived from
[CleverRaven/Cataclysm-DDA](https://github.com/CleverRaven/Cataclysm-DDA).
Cataclysm game content is licensed under the Creative Commons
Attribution-ShareAlike 3.0 Unported license, subject to the additional
third-party notices contained in the game's `LICENSE.txt` and related license
files.

The independent data-generation repository was designed for this edition with
reference to the public data format documented by
[nornagon/cdda-data](https://github.com/nornagon/cdda-data). Generated records
retain source file paths back to the corresponding Cleanwater Bomb release.

## Interface translations and third-party packages

Interface translations are loaded from the original guide's public Transifex
project and remain credited to its translation contributors. Game text
translations instead come directly from Cleanwater Bomb release sources.

Third-party JavaScript packages, fonts, and other assets retain their own
copyrights and licenses as declared by those packages and source files. The
repository's `LICENSE` covers the guide application; it does not replace any
applicable third-party license.
